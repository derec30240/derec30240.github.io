本站最初于2025年7月15日使用 GitHub Pages 搭建。主要用于记录自己平时的一些技术笔记和踩坑经验，偶尔也会随手写点生活随笔和杂谈。

技术实现
#

网站托管在 GitHub Pages 上，日常更新流程如下：

  • 本地撰写、编辑内容
  • 提交到 GitHub 的一个 Private 仓库
  • 通过 GitHub Actions 自动打包生成静态网页
  • 自动推送到 Public 仓库中
  • 由 GitHub Pages 将生成的静态网页投放到公网
